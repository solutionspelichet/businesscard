Votre QR Code

✅ Sur Android (Chrome, Edge...)
  • Ouvrez cette page dans votre navigateur.
  • Appuyez sur les trois points (en haut à droite).
  • Choisissez « Ajouter à l'écran d'accueil ».
  • Nom : ex. « QR Pelichet ».
  • Appuyez sur « Ajouter ».

🔗 Si vous voulez afficher uniquement l’image, utilisez le lien direct : qr.png.

🍏 Sur iPhone (Safari uniquement)
  • Ouvrez cette page dans Safari.
  • Appuyez sur l’icône Partager (rectangle avec flèche vers le haut).
  • Choisissez « Sur l’écran d’accueil ».
  • Nom : ex. « QR Pelichet ».
  • Appuyez sur « Ajouter ».
